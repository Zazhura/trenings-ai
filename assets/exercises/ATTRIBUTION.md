# Exercise Animation Attribution

This directory contains Lottie animation files for exercise demonstrations.

## Placeholder Animations

**Note:** These are placeholder animations. In production, replace with properly licensed Lottie animations from:
- LottieFiles (https://lottiefiles.com)
- IconScout (https://iconscout.com)
- Or create custom animations

## Required Attribution Format

For each animation file, document:
- **File name**: `exercise_name_side.json`
- **Source**: [URL to source]
- **License**: [License type, e.g., MIT, CC-BY, etc.]
- **Author**: [Author name if available]

## Example Entry

```
### pushup_side.json
- **Source**: https://lottiefiles.com/animations/pushup-side-view
- **License**: MIT
- **Author**: Exercise Animations Co.
```
